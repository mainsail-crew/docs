# Slicers

Your slicer requires specific configuration to work optimally with Mainsail and Klipper.
Proper setup enables features that enhance your printing experience.

## Features Requiring Slicer Configuration

- **Thumbnails** – Display preview images in the file browser and print dialogs
- **Exclude Objects** – Cancel individual objects during a print without stopping the job
- **Layer Information** – Enable layer count display and "Pause at Layer" functionality
- **Direct Upload** – Send G-code files directly from your slicer to the printer
- **G-code Metadata** – Show filament type, print time estimates, and other details
